## üõ†Ô∏è Abrindo e fechando arquivos

Em Python, usamos a fun√ß√£o `open()` para abrir arquivos:

```python

arquivo = open("exemplo.txt", "r")

# Lendo o conte√∫do

print(conteudo)

# Fechando o arquivo
arquivo.close()
```
## üìå Boa pr√°tica:

Sempre fechar arquivos para liberar recursos.
Alternativa mais segura: usar o context manager (with):

```python

    conteudo = arquivo.read()
    print(conteudo)
# arquivo √© automaticamente fechado ao sair do bloco
```

## ‚úçÔ∏è Escrevendo em arquivos

* Modos de abertura:

    - 'w': escrita (sobrescreve)

    - 'a': acrescentar ao final

    - 'x': cria novo arquivo (erro se existir)

```python

# Escrevendo no arquivo
with open("saida.txt", "w") as arquivo:
    arquivo.write("Linha 1\n")
    arquivo.write("Linha 2\n")
# Acrescentando ao arquivo existente
with open("saida.txt", "a") as arquivo:
    arquivo.write("Linha 3\n")
```

## üìä Manipulando arquivos CSV

CSV (Comma Separated Values) √© usado para armazenar dados tabulares.

```python
import csv

# Lendo CSV
with open("dados.csv", newline="") as csvfile:
    leitor = csv.reader(csvfile, delimiter=",")
    for linha in leitor:
        print(linha)

# Escrevendo CSV
with open("saida.csv", "w", newline="") as csvfile:
    escritor = csv.writer(csvfile)
    escritor.writerow(["Nome", "Idade", "Cidade"])
    escritor.writerow(["Ana", 25, "S√£o Paulo"])

```
