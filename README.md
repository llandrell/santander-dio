## 📋 Sobre o Projeto

Este repositório documenta minha jornada no **Bootcamp Santander 2025 - Back-End com Python**, uma parceria entre Santander e [Digital Innovation One (DIO)](https://dio.me). Funciona como um caderno de estudos digital onde registro aprendizados, resolvo desafios e desenvolvo projetos práticos.

### 🎯 Objetivos do Bootcamp

O programa visa capacitar desenvolvedores para o mercado de back-end, abordando:

- **Fundamentos Python**: Sintaxe, estruturas de dados e programação orientada a objetos
- **Desenvolvimento Web**: Criação de APIs REST com FastAPI
- **Banco de Dados**: Integração com bancos relacionais (SQL)
- **Boas Práticas**: Versionamento com Git, testes e documentação
- **Desenvolvimento Profissional**: Networking e preparação para o mercado

## 🔍 Conteúdo por Seção

### 📚 Anotações
Resumos teóricos e conceitos fundamentais organizados por módulo, cada arquivo contém:

- **`1-tipos_operadores.md`** - Tipos básicos, operadores aritméticos, relacionais, lógicos e de atribuição em Python
- **`2-estruturas_condicionais.md`** - Estruturas condicionais (if, else, elif) para controle de fluxo
- **`3-manipulando_string.md`** - Manipulação de textos, fatiamento, concatenação e métodos de string
- **`4-lista_python.md`** - Estruturas de dados: Listas e suas operações
- **`5-truplas.md`** - Estruturas de dados: Tuplas e suas características
- **`6-conjuntos.md`** - Estruturas de dados: Conjuntos (Sets) e operações
- **`7-dicionario.md`** - Estruturas de dados: Dicionários e manipulação
- **`8-funcoes.md`** - Criação e uso de funções, parâmetros, retorno e escopo

> 📝 **Cada arquivo inclui:** Explicações teóricas, exemplos práticos de código e boas práticas de programação

### 💻 Desafios
Exercícios práticos organizados por módulo para fixação dos conceitos:

- **`1-primeira_Aula/`** - Primeiros exercícios e introdução às estruturas for
- **`2-Operadores/`** - Exercícios com operadores Python e sistema de CNH
- **`2-conhecendo_python/`** - Fundamentos e conceitos básicos da linguagem
- **`4-estruturas-condicionais-repeticao/`** - Estruturas de controle e loops
- **`5-manipulando_string/`** - Desafios de manipulação e fatiamento de strings
- **`6-listas_python.py`** - Exercícios práticos com estruturas de lista

### 🚀 Projetos
Aplicações práticas desenvolvidas durante o bootcamp:

- **`dio-lab-open-source/`** - Laboratório prático de Git e GitHub para colaboração em projetos open source
- **`sistema-bancario/`** - Simulação de operações bancárias com cadastro de clientes e transações
- **`system-bank.v2/`** - Versão aprimorada do sistema bancário com novas funcionalidades

> 📋 **Cada projeto inclui:** README próprio, código-fonte organizado, documentação técnica e instruções de execução

```bash

cd santander-dio
```

### Navegando pelo Conteúdo
- **`/anotacoes`** - Explore para revisar conceitos teóricos organizados por módulo
- **`/desafios`** - Analise soluções de exercícios práticos e algoritmos
- **`/projetos`** - Examine projetos práticos e aplicações reais
- Cada seção possui seu próprio `README.md` com instruções específicas

### Utilizando o Repositório
O repositório está estruturado para facilitar o aprendizado:

**📚 Para Estudos Teóricos:**
- Acesse `/anotacoes` para conceitos organizados progressivamente (1-8)
- Cada arquivo contém resumos, exemplos práticos e referências

**💻 Para Prática de Algoritmos:**
- Explore `/desafios` para exercícios organizados por módulo
- Soluções práticas desde conceitos básicos até estruturas avançadas
- Exercícios progressivos: operadores → strings → listas → estruturas condicionais

**🚀 Para Projetos Reais:**
- Consulte `/projetos` para aplicações práticas completas
